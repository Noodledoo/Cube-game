# Cube Game

A 3D cube runner game built with Python and Pygame. Navigate your spinning cube through obstacles, collect points, and survive as long as you can.

## Controls

- **Arrow Keys / WASD** - Move left and right
- **Space / Up / W** - Jump
- **ESC** - Pause / Quit
- **R** - Restart (after game over)

## How to Play

- Dodge the red/orange/purple obstacles coming from the right
- Collect yellow spinning cubes for bonus points (+100 each)
- Score increases over time as you survive
- Speed gradually increases, making the game harder

## Setup

```bash
pip install -r requirements.txt
python cube_game.py
```

## Requirements

- Python 3.8+
- Pygame 2.5+
